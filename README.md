## ✦ Overview

A premium personal portfolio built with **vanilla HTML, CSS, and JavaScript** — no heavy frameworks, just precise control over every pixel, transition, and animation. The site showcases my work, philosophy, and personality through a cinematic, detail-driven UI.

## ✦ Tech Stack

**Core**
- Plain HTML5, CSS3, and ES Module JavaScript
- [Vite 6](https://vitejs.dev/) — dev server & build tool

**Animation & 3D**
- [GSAP 3](https://gsap.com/) — scroll-driven animations, timeline sequences
- [Three.js](https://threejs.org/) — 3D metallic cube & dissolve overlay
- [Lenis](https://lenis.darkroom.engineering/) — smooth scroll
- [Cobe](https://cobe.vercel.app/) — interactive WebGL globe

**Typography**
- Google Fonts: Anton, Bebas Neue, DM Sans, DM Serif Display, Fraunces, Inter, Montserrat, Playfair Display

**Deployment**
- [Vercel](https://vercel.com/) — with custom rewrite rules via `vercel.json`

## ✦ Design Highlights

- 🎨 **Bento grid layout** — identity, philosophy, contact, project showcase & globe cards
- 🖱️ **Custom cursor** with radial mouse glow
- 🐱 **Hero cat** with tracking pupil eyes that follow the cursor
- 🌐 **Interactive WebGL globe** — locale-aware, draggable
- 🧊 **Draggable 3D metallic cube** rendered with Three.js
- 🕐 **Live analog clock** and real-time IST clock in the identity card
- ⌨️ **Command palette** — keyboard-accessible search overlay for navigation
- 📧 **Tap-to-copy email** with animated clipboard feedback
- 🌊 **Cinematic sticky-stack scroll transition** between sections
- 🎵 **Jamify page** — live Spotify listening activity
- 📱 Fully responsive with a dedicated mobile nav pill

## ✦ Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The dev server runs at `http://localhost:5173` by default.
